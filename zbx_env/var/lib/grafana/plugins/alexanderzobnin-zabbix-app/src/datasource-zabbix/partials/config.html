Data Source Select Data Source for Zabbix history database. In order to use this feature it should be created and configured first. Zabbix plugin uses this data source for querying history data directly from the database. This way usually faster than pulling data from Zabbix API, especially on the wide time ranges, and reduces amount of data transferred.
